# ktCandyCrash

．使用Kotlin仿照Candy Crash製作。 <br/>
．運用Firebase實現第三方登入認證，簡訊OTP註冊認證。<br/>
．動態產生Button物件，並賦予屬性及動畫。<br/>
．具有簡易計分計時功能。<br/>
